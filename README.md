# Automated Web Testing

## Description
  This project utilises Cucumber, a bahviour-driven development, language to test a website using the Selenium testing library. We model each webpage using a PAge Object Model (POM).   

## Features
 List of features:  
 - Cucumber scripts
 - Selenium
 - Hamcrest

## Prerequisites
 To run this project, the following technologies are required:
 - Java (Version 23)
 - Maven
 - Junit (Version 3.8.1)
 - Selenium (Version 4.22.0)
 - Cucumber (7.18.0)
 - Hamcrest (2.2)
